# Източници — BG България (reference)

Дата на проверката: 2026‑07‑18. Статус на connector: active (external Scheduled Task).

## Официални портали
- **eufunds.bg** — Единен информационен портал за фондовете на ЕС в България.
- **ИСУН / eumis2020** — информационна система за управление и наблюдение.
- **esf.bg** — Европейски социален фонд (ОПРЧР / ПРЧР).
- **az.government.bg** — Агенция по заетостта (мерки по заетост/младежка заетост).
- **ПКИП** — Програма „Конкурентоспособност и иновации в предприятията“.

## Управляващи органи
Управляващите органи по оперативните програми (ПРЧР, ПКИП, Околна среда, Образование,
ПНИИДИТ и др.) — през съответните секции на eufunds.bg / ресорните министерства.

## Технически формат / покритие / ограничения
- Настоящите BG данни се поддържат външно (Claude Scheduled Task пише в D1 `projects`).
- Покритие: активни и предстоящи национални процедури с фокус младежка заетост + общи.
- Ограничение: бюджетът е свободен текст (не се сумира), поле „сектор“ липсва —
  ползва се програмата като измерение.
